How to run it?

You can run this project using one of following options:
  1. Serve it from localhost using XAMPP, just copy dist folder into htdocs folder
  2. Serve it from localhost using node-static:
    • You will need NodeJS, download it from [here]
    • NPM come along with node.js
    • Install node-static using 
      $ npm install --global node-static
      Once node-static has been installed, navigate to dist folder and run 
      $ static

How to develop?

This project have been developed using Yeoman's automation workflow. Therefore, in order to extend it for your need, some basic knowledge about node.js, npm, bower and Yeoman is required.

You can read full documentation for each of them from their sites

  • node.js's docs
  • Learn Yeoman
  • NPM's docs
  • Bower's docs

In this document's span, I will show you how to setup the environment and get the project run properly.

  1. First, install node.js from [NodeJS's home page] if you haven't had it yet. NPM come along with node.js
  2. Next, you'll need yeoman and bower installed, use commands below 
    npm install --global yo
    npm install --global bower
  3. In order to get project running, you have to install its dependencies, move to project directory and run 
    npm install
  4. Caution:

    Some of you probably already got used to Yeoman's workflow and wonder why I didn't say anything about using bower install command.

    This theme's released with bower_components, so you dont need to do that.

    Well, you can install these components explicitly if you want though it's not recommended. By the time you bought this theme, these dependencies probably has been updated, so mass update may cause unwanted conflicts. Use it with caution.

  5. Once dependencies have been installed, start previewing by running 
    grunt serve
  6. To build project, run 
    grunt build
